# 暮らしのカルテ / 洗濯・水回り診断 — デザイントークン（実装前に確定・自己批評済み）

案件固有の視覚アイデンティティ。「暮らし・住まいの水環境を、一枚のやさしい診断票で記録する」
総合メディアの洗濯・水回りサブ診断。核は **やわらかい水と泡の世界観 ×「暮らしの診断票（カルテ）」の記録性**。
姉妹サイト karada-carte（深緑 #1E3A34 × 用紙 × 印泥朱 × 明朝 × Georgia）とは
**色・書体・モチーフを完全に別系統**にし、共有するのは構造（config駆動・GAS・インラインSVG・GEO）のみ。

## コンセプト（1行 ＝ 全工程の統一基準）
> 洗濯・水回りの「なんとなくの不快」を、水滴のゲージと一枚のやさしい診断票にして、
> あなたの暮らしの水環境を継続的に見守る。（比較しない・並べない・あなたの数字を発行する）

## やらないことリスト（compliance-qa の差し戻し基準）
- 洗浄力・消臭・除菌の**効果を断定/誇大**にしない（景表法・薬機法）。「〜な方に選ばれています」等の体験ベース。
- 「洗剤の代わりになる」等の断定をしない。マグネシウム洗濯は実証範囲・体感の目安に留める。
- before/after、他社比較、ランキング演出をしない（＝並べない・比較しない）。
- AIデフォルト（水色一辺倒・空グラデ箱・意味のない01/02連番・ダークネオン・絵文字アイコン乱用）を回避。
- 過剰アニメ・自動再生・重い演出をしない（`prefers-reduced-motion` 尊重）。
- karada-carte の緑×朱印×明朝の視覚をリサイクルしない。

## シグネチャー要素（記憶される唯一の独自要素）
**「暮らしの水環境カルテ（診断票）」の発行体験。** 診断の最後に、カルテ番号・発行日・
**水環境スコア（水滴が満ちるゲージ / 0–100）**・タイプ判定の**まる型シール（珊瑚色の“やさしい判子”）**・
所見（傾向の目安）・洗剤過敏度サブ指数を持つ一枚の“やわらかい診断票”が発行される。
karada の「印泥朱の判子」に対し、こちらは**手描き水滴のまるシール**＝可愛く親しみやすい確定の印。
周囲は淡い水・泡で静かに保ち、確定の瞬間（シールの押印・スコアの充填）に華やぎを一点集中させる。

## カラー（名前付き・6色＋3機能色 ／ AIデフォルト回避）
「信頼のディープティール × 清潔な泡白 × 可愛い珊瑚のアクセント × マグネシウムの若葉」。
水色一辺倒を避け、**深い青緑（信頼）を主構造**に、**暖色の珊瑚（可愛さ・確定の印）**を一点差し色、
**若葉ミント（マグネシウム＝自然由来・環境配慮）**を意味アクセントに据える。

| 変数 | HEX | 名前 | 役割 |
|---|---|---|---|
| `--marine`  | `#0E3A47` | ディープティール | 主構造色（ヘッダ/フッタ/濃地/見出し）＝水×信頼。karadaの深緑と別系統の青緑 |
| `--tide`    | `#2C7E92` | タイドブルー | 中間ティール。帯・グラデ・ゲージ水面 |
| `--aqua`    | `#DCEEF0` | 泡アクア | 淡い水面。カードの地・washes |
| `--foam`    | `#F7FCFC` | フォームホワイト | ページ地色＝泡白。冷たく清潔（クリームでも純白でもない＝AIデフォルト回避） |
| `--ink`     | `#1F2A2E` | 墨藍 | 本文。純黒を避けた青みの濃炭 |
| `--muted`   | `#67787D` | 霧グレー | 注記・副次テキスト |
| `--line`    | `#DCE8E7` | 水際罫 | カルテの薄い罫線（1px） |
| `--coral`   | `#F4795B` | 珊瑚（さんご） | **唯一の暖色アクセント**。まるシール・スコア針・確定CTA・可愛い差し色 |
| `--mint`    | `#74B79B` | 若葉ミント | マグネシウム＝自然由来/環境配慮。タグ・安心の微アクセント |
| `--sun`     | `#F7C948` | 陽だまり | ごく少量のきらめき（泡・結果票の紙吹雪・☑）＝“可愛く派手”の火種 |

機能色：`--ok #2F8F6B` / `--danger #C4553C` / `--warn #E0A43B`

## タイポグラフィ（和文・実在フォント／外部CDN無し・システム同梱で完結）
可愛さ＝**丸ゴシック**、信頼＝**UD書体**、数字＝**丸みのあるサンセリフ**で「可愛い×診断書の精度」を両立。
外部フォント配信はセキュリティベースライン（外部JS/CDN回避）に反するため、**システム同梱**で構成。

- 見出し（Display / 診断票タイトル）:
  `"Hiragino Maru Gothic ProN","BIZ UDPGothic","Yu Gothic UI","Meiryo",sans-serif`
  — 丸ゴの親しみ。Windowsは BIZ UDPGothic / Yu Gothic UI に落ちる（全て同梱・外部配信なし）
- 本文（Body）:
  `system-ui,"Hiragino Kaku Gothic ProN","BIZ UDPGothic","Yu Gothic UI","Meiryo",sans-serif`
- 数値（スコア/カルテ番号）:
  `"Trebuchet MS","Verdana",system-ui,sans-serif` ＋ `font-variant-numeric: tabular-nums`
  — 丸みのある等幅数字。karada の Georgia（明朝系セリフ）と意図的に別系統。ラテン小ラベル
  （WATER CARE RECORD 等）は字間を広げて“票”の記名性を出す

## レイアウト
- 母屋（`/`）: 暮らし・住まいの**総合メディアハブ**。「いま使える診断」1枚を主役カード＋
  「これから広げる領域」タイル群（掃除・防臭・水回り・室内干し…）。config.json の
  コンテンツ一覧で無限増設。max 960px。
- `/sentaku/`: ヒーロー→GEO直接回答（洗剤×肌荒れ/部屋干し臭/洗濯槽カビの原因と対策）→
  **7問診断**→**水環境カルテ票発行（水滴ゲージ・まるシール・所見）**→タイプ別マグちゃんアプローチ→
  複合型の併用提案→オファー（定期/らくトク便）→LINE・再診断案内→しくみ→FAQ→免責。max 880px。
- `/mizu-kankyo-score/`: 「暮らしの水環境スコア／洗剤過敏度」の定義・算出根拠の解説（GEO一次引用の対象）。
- 診断票は水際罫・カルテ番号・水滴ゲージ・まるシール・所見欄を持つ“紙面”。印刷/保存に耐える print CSS。

## ビジュアル / 素材レイヤー（可愛く派手に・空グラデ箱禁止）
全て**ローカル同梱**（インラインSVG＋`assets/`）。外部ホットリンク皆無（誤写真・著作権の教訓）。
- 透かし背景：淡い水・泡・葉・水滴のモチーフSVG（`--aqua`/`--mint`の低透明）でセクション地に世界観。
- ヒーロー：手描き水彩調の「清潔な洗濯・やさしい水・赤ちゃん・ナチュラルな暮らし」原画SVG（オリジナル）。
- 4タイプの可愛いアイコン（部屋干し臭・敏感肌ベビー・食器水回り・お風呂）をSVGで統一トーン。
- 写真は将来 `images.css` の変数スロットで1ファイル差し替え可（Claude Design生成/撮影を後差替）。
  ＝“派手さ”は現状オリジナルSVG＋淡い透かし＋アニメで担保し、写真は上乗せ枠として用意。

## モーション（subject に奉仕・抑制）
`prefers-reduced-motion` 尊重。ステップ切替の微フェード、**水環境スコアの一度きりの充填カウントアップ**、
**まるシールの押印（scale＋opacity 一回）**、スクロール連動フェードイン、泡のごく緩やかな浮遊。
パララックス強・自動再生・過剰演出はしない。

## 品質フロア
タップ44px/フォント16px、キーボードfocus可視（outline）、色コントラストAA、
CSP(meta)+frame-busting、画像は全てローカル（外部ホットリンク皆無）、レスポンシブ完全対応。

## 自己批評（AIデフォルト検査 — 実装前ゲート）
- [x] ヘルスケア/水回りにありがちな**水色一辺倒**？→ **不採用**。主構造は深い青緑#0E3A47、
      差し色は暖色の珊瑚#F4795B、意味色に若葉ミント。水色は淡い地色(--aqua)に限定。
- [x] 空グラデ箱・意味のない大数字ヒーロー？→ **不採用**。ヒーローは手描き水彩調の原画SVG。
      数字は結果票の中で「水環境スコア」＝意味に根ざして初めて大きく出す。
- [x] 意味のない 01/02/03 連番？→ 使わない。ステップは「STEP n/7」で機能的にのみ。
- [x] ダークネオン/絵文字アイコン乱用/影カード乱用？→ 回避。淡い水地×水際罫×珊瑚1点。
- [x] karada-carte（緑×朱印×明朝×Georgia）の視覚リサイクル？→ **全別系統**（青緑×珊瑚×ミント／
      丸ゴ×Trebuchet／水滴ゲージ＋まるシール）。共有は構造（config駆動・GAS・SVG・GEO）のみ。
- [x] マグちゃんブランド（magchan.com）と意味接続？→ 若葉ミント＝マグネシウム/自然由来/環境配慮、
      泡白＝洗剤に頼らない清潔感、水滴＝水回り全般。ブランドの訴求軸（肌にやさしい・環境配慮）と一致。
